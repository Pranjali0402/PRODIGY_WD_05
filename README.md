# Weather Web Application

## Project Overview
This project was developed as Task 5 of my Web Development Internship at Prodigy InfoTech. It is a weather web application built using HTML, CSS, and JavaScript that provides real-time weather information for any city using the OpenWeather API. The application displays essential weather details through a responsive and easy-to-use interface.

## Features
- Search weather by city name
- Real-time weather information
- Displays temperature, humidity, and wind speed
- Weather condition icons
- Responsive and user-friendly interface

## Technologies Used
- HTML5
- CSS3
- JavaScript
- OpenWeather API

## How to Run
1. Download or clone this repository.
2. Open the project folder.
3. Add your OpenWeather API key in the JavaScript file if required.
4. Open `index.html` in any modern web browser.
